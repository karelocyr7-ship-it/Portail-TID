# Portail TAD Groupe

Portail centralisé des applications métiers de TAD Groupe.

Adresse cible :

https://portail.tadgroupe.com

Le projet comprendra notamment :

- le portail Web ;
- l'authentification Keycloak ;
- le catalogue des applications ;
- les rôles et autorisations ;
- les agents Codex nocturnes ;
- les procédures de sauvegarde et d'exploitation.
