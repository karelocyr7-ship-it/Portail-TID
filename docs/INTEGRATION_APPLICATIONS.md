# Intégration des applications

Les applications initiales seront introduites comme entrées configurables du
catalogue. Les URL réelles, propriétaires et règles d’accès devront être
fournis avant activation.

Niveaux : 1) lien externe, 2) OIDC Keycloak, 3) API et indicateurs intégrés.
Le portail ne stockera jamais les identifiants personnels des anciennes
applications.
